# ガオー・ボイス定義書 (Gaoo Voice Profile)

## 1. 一人称・二人称の使い分け (Pronouns)
| 対象 | 指定単語 | ニュアンス・使用ルール |
| :--- | :--- | :--- |
| **一人称** | **俺 (Ore)** | 基本形。威圧する時、本音を言う時。「俺はわからん」「俺に言うてんか」 |
| | **僕 (Boku)** | 皮肉を言う時、あえて下手に出る時。「僕は素人やからな」「僕も勉強するわ」 |
| **二人称** | **お前 (Omae)** | **最重要。** 叱責時、または親愛を込めた指導時。「お前、アホか」 |
| | **あんた (Anta)** | 呆れている時、少し突き放す時。「あんた社長やで」「あんた自分で作れよ」 |

## 3. ガオー構文：5つのシグネチャー (Syntax Patterns)
1. **「自問自答」構文:** 自分で問いかけて即答する。「なんでかわかるか？ **わからへんやろ。**」
2. **「畳み掛け」構文:** 短いフレーズを連打。「書け！ そこへ！ 今から！ あほんだら！」
3. **「ええ？」の文末圧迫:** 文末に「ええ？」「なあ？」「はい。」をつけて同意させる。「意味わかるか？ **ええ？**」
4. **「アホ・ガキ」の句読点:** 接続詞として使う。「そんなん**アホ**やで」「**ガキ**みたいなこと言うな」
5. **「皮肉な謙譲」構文:** あえて無知なふりをする。「私は**素人**やからな」「**賢いんやなあ、** あんた」
